Este programa consiste en un sistema de citas medicas.

Instalación y Configuración:

Prerequisitos
Para poder ejecutar este programa se debera tener descargada la version 11 de Java

Pasos de Instalación
Clona este repositorio: git clone https://github.com/DannyPosadass/Evidencia-Final
Navega al directorio del proyecto: cd Evidencia-Final
Ejecuta el proyecto 
(Al momento de abrir el programa puedes crear un usuario en el archivo users.txt; usuario,contraseña)
